# 📊 Dashboard-Vendas-Xbox2


## 📘 Visão Geral
Este projeto foi desenvolvido como parte de um desafio prático com o objetivo de criar um **dashboard de vendas em Excel**.  
A proposta é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho comercial e apoiando a tomada de decisões baseadas em dados.

## 📂 Estrutura do Repositório
- `README.md` → documentação do projeto  
- `Dashboard_Vendas.xlsx` → arquivo Excel com o dashboard concluído  
- `/data` → dados brutos utilizados (opcional)  

---

## 📊 Dados Utilizados
- Fonte: planilha fictícia de vendas  
- Estrutura: colunas com **Data, Produto, Categoria, Quantidade, Valor Total**  
- Período: dados mensais de vendas  

---

## 🖥️ Funcionalidades do Dashboard
- **Filtros dinâmicos** por período, produto e categoria  
- **KPIs principais**:
  - Faturamento total  
  - Quantidade vendida  
  - Ticket médio  
- **Gráficos interativos**:
  - Linha (tendência de vendas ao longo do tempo)  
  - Barra (comparativo por produto/categoria)  
  - Pizza (participação por categoria)  

---

## 🚀 Como Reproduzir
1. Baixe o arquivo `Dashboard_Vendas.xlsx` do repositório.  
2. Abra no **Microsoft Excel** (versão 2016 ou superior recomendada).  
3. Explore os filtros e gráficos para visualizar os resultados.
